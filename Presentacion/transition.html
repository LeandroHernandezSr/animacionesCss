Transition

Conceptos básicos de las transiciones en CSS

Propiedad transition

La propiedad CSS `transition` permite crear efectos de transición suaves cuando se cambian las propiedades de un elemento. Podemos especificar qué propiedades deben tener una transición, la duración de la transición y la función de temporización utilizada.

transition: [propiedades] [duración] [función de temporización]

Funciones de temporización

Las funciones de temporización especifican cómo se interpolan los valores durante una transición. Algunas de las funciones comunes incluyen `ease`, `linear`, `ease-in`, `ease-out`, `ease-in-out` y `cubic-bezier`.

.nombre_de_clase{
transition-timing-function: ease;
}
