Animation

Conceptos básicos de la animación en CSS

Propiedad animation

CSS proporciona la propiedad animation para controlar la animación de un elemento. Podemos definir la duración, la función de temporización, los retrasos y otros parámetros.

animation: [animacion a utilizar] [duracion de la animacion] [nº repeteciones];

Regla keyframe

Los keyframes son puntos clave en una animación donde especificamos cómo se ve el elemento en ese momento específico. Podemos definir varios keyframes para crear una animación más compleja.

@keyframes nombre-animacion {
0% {
/* Propiedades CSS en el inicio de la animación */
}
50% {
/* Propiedades CSS a la mitad de la animación */
}
100% {
/* Propiedades CSS al final de la animación */
} }

Funciones de temporización

Además de las animaciones basadas en keyframes, CSS también admite transiciones suaves entre los estados de los elementos. Podemos definir propiedades como transition-property, transition-duration y transition-timing-function para controlar las transiciones.

.nombre_de_clase{
animation-timing-function: linear;
}
